
Here are several scripts that use Zededa's intermediate API to control Fortiflex licenses


Intermediate API https://zedcontrol.orchid-d3vw.com/api/v1/firewall/serial/
talks to Fortinet's Fortiflex API https://support.fortinet.com/ES/api/fortiflex/v2

Official doc is here: https://fndn.fortinet.net/index.php?/fortiapi/954-fortiflex/956/
Unofficial doc is here: https://github.com/movinalot/fortinet-flexvm
Zededa's doc is here: https://zededa.atlassian.net/wiki/spaces/SCS/pages/2107539464/Zededa+s+Intermediate+Firewall+API+for+VW


If a fortigate license ever was stopped, it can not work until reactivated.
if a fortigate license's token was renewed, this license can be successfully installed on a new VM with the new token.
